# Personal Portfolio

A responsive personal portfolio website showcasing projects, skills, and experience with a contact form that stores messages in Google Sheets.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Project Showcase**: Display of completed projects with descriptions
- **Skills Section**: Highlighting technical and professional skills
- **Contact Form**: Integrated contact form that stores messages in Google Sheets
- **Smooth Navigation**: Intuitive navigation and user experience

## Technologies Used

- **HTML5**: Structure and semantic markup
- **CSS3**: Styling, animations, and responsive design
- **JavaScript**: Interactivity, form validation, and Google Sheets integration
- **Google Sheets API**: Backend for storing contact form submissions

## Getting Started

1. Clone or download the project
2. Open `index.html` in a web browser
3. Navigate through sections to view portfolio content
4. Use the contact form to send messages (stored in Google Sheets)

## Contact Form Integration

The contact form is integrated with Google Sheets to automatically store and organize all incoming messages:

- Form data is captured and sent to a Google Sheet
- Each submission creates a new row with timestamp, name, email, and message
- Easy to access and manage all contact inquiries

## How to Use

1. Fill out the contact form with your name, email, and message
2. Click submit
3. Your message will be stored in the connected Google Sheet
4. Confirmation message will appear on successful submission

## Installation

No installation required. Simply download the files and open `index.html` in any modern web browser.
